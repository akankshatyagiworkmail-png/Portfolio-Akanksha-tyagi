Case Study

SKDCRM —
AI-Built Sales CRM

A fully functional, production-deployed sales CRM built entirely using AI — with zero traditional coding background. Conceived, designed, and shipped to solve a real business problem at SKD Propworld Pvt. Ltd.

100% AI-Built Claude AI (Anthropic) Supabase DB Live on Vercel 3 Role Portals
View Live App ↗
The Problem — Before SKDCRM
  • No system to track incoming leads and enquiries — calls, WhatsApp, walk-ins with no central log
  • Sales team had zero visibility into follow-up status or lead pipelines
  • Critical follow-ups were missed daily — leads slipping through the cracks
  • Management had no data to make decisions — no reports, no dashboards, no accountability
  • No way to track executive activity, call outcomes, or lead assignments
The Solution

SKDCRM is a browser-based, mobile-friendly web application with three separate role-based portals — Admin, Manager, and Executive — each showing only what's relevant to that user's work. It's live, actively used, and built entirely through AI-assisted development using Claude AI and Supabase as the real-time database backend.

Architecture — 3 Role-Based Portals
Admin
Full system control and oversight
  • Manage all users & roles
  • View all leads & pipeline
  • Bulk lead uploads
  • System configuration
  • Full reports access
Manager
Team management and performance
  • Assign leads to executives
  • Track team performance
  • Monitor call logs
  • View executive locations
  • Manager-level reports
Executive
Daily sales execution
  • View assigned leads
  • Log calls & outcomes
  • Update lead status
  • WhatsApp messaging
  • Location check-in
